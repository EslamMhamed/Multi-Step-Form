##  Technologies Used

-  React
-  Tailwind CSS
-  TypeScript
-  `useReducer` for state management
-  Context API for sharing global state
-  `react-router-dom` for step navigation

## Getting Started
 To run the project locally:

 git clone https://github.com/EslamMhamed/Multi-Step-Form.git

 cd Multi-Step-Form

 npm install

 npm run dev
